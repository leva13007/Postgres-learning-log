# Resources

## Official Documentation

- [PostgreSQL 18 Official Docs](https://www.postgresql.org/docs/18/)
- [PostgreSQL Wiki](https://wiki.postgresql.org/)
- [PostgreSQL Tutorial (official)](https://www.postgresql.org/docs/18/tutorial.html)
- [SQL Language Reference](https://www.postgresql.org/docs/18/sql.html)

## Books

- *PostgreSQL: Up and Running* — Regina Obe, Leo Hsu
- *The Art of PostgreSQL* — Dimitri Fontaine
- *PostgreSQL 14 Internals* — Egor Rogov (freely available)
- *Learning SQL* — Alan Beaulieu

## Courses & Tutorials

- [PostgreSQL Exercises](https://pgexercises.com/) — free interactive SQL exercises
- [SQL Teaching](https://www.sqlteaching.com/) — interactive SQL tutorial
- [Mode SQL Tutorial](https://mode.com/sql-tutorial/) — practical SQL lessons
- [PostgreSQL Tutorial](https://www.postgresqltutorial.com/) — comprehensive tutorial site

## Tools

- [psql](https://www.postgresql.org/docs/18/app-psql.html) — built-in interactive terminal
- [pgAdmin](https://www.pgadmin.org/) — GUI administration tool
- [DBeaver](https://dbeaver.io/) — universal database client
- [pgcli](https://www.pgcli.com/) — psql alternative with auto-completion
- [EXPLAIN Visualizer](https://explain.dalibo.com/) — query plan visualization

- [Citus Data Blog](https://www.citusdata.com/blog/) — performance and scaling articles
- [pganalyze Blog](https://pganalyze.com/blog) — monitoring and optimization
- [Use The Index, Luke](https://use-the-index-luke.com/) — SQL indexing and tuning
