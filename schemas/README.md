# Schemas

Database schemas and ER diagrams for learning projects.

## Format

Diagrams use [Mermaid](https://mermaid.js.org/syntax/entityRelationshipDiagram.html) syntax for version-control-friendly ER diagrams that render directly in GitHub/VS Code.

## Naming Convention

`<project-name>.md` — e.g. `bookstore.md`, `blog-platform.md`

Use [schema-template](../templates/schema-template.md) to create new entries.
