# Project Structure

## Root Documents

README.md - What is this project about?
ROADMAP.md - What are the future plans for this project?
PROGRESS.md - What has been accomplished so far?
RESOURCES.md - Curated list of learning resources: official docs, books, courses, tools, articles.
glossary.md - Dictionary of PostgreSQL terms with definitions and related topics.

## Learning Directories

sessions/ - Detailed diary of learning sessions, dated and structured. Each session includes goals, key concepts, SQL snippets, challenges faced, and takeaways. Provides insights into the learning journey.

notes/ - Various notes, tips, and concept summaries found useful during the learning process. Includes links to helpful articles, tutorials, and any other beneficial information.

examples/ - Practical examples and mini-projects. Each example includes a description, the code, and explanations of how it works. Meant to show how concepts are applied in real-world scenarios.

mistakes/ - Documentation of mistakes made during the learning journey. Each entry includes a description of the mistake, what was learned from it, and how it was corrected. Helps others avoid similar pitfalls.

## Reference Directories

cheatsheets/ - Quick reference sheets for daily work: SQL syntax, psql commands, data types, common patterns. Designed for fast lookup.

exercises/ - Practice tasks organized by topic with solutions. Each exercise includes a task description, schema setup, hints, and a collapsible solution with explanation.

schemas/ - Database schemas and ER diagrams (Mermaid format). Visual representations of database structures for learning projects.

queries/ - Organized library of useful SQL queries by category. Each query includes its purpose, the SQL code, explanation, and performance notes.

## Infrastructure

src/ - Docker configuration, SQL initialization scripts, and shell automation scripts for setting up and managing the PostgreSQL learning environment.

## Media

youtube/ - Media assets for YouTube content about the PostgreSQL learning journey. Includes thumbnails, screenshots, and other graphics.

## Meta

templates/ - Templates for quickly generating new learning sessions, notes, examples, mistakes, cheatsheets, exercises, schemas, and query entries. Designed to streamline documentation and ensure consistency.
