# Cheatsheets

Quick reference sheets for daily PostgreSQL work.

## Naming Convention

`<topic>.md` — e.g. `data-types.md`, `psql-commands.md`, `joins.md`

## Suggested Topics

- SQL syntax basics (SELECT, INSERT, UPDATE, DELETE)
- psql meta-commands (`\dt`, `\d`, `\l`, etc.)
- Data types overview
- String functions
- Date/time functions
- Aggregate functions
- Index types
- Constraints

Use [cheatsheet-template](../templates/cheatsheet-template.md) to create new entries.
