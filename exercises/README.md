# Exercises

Practice tasks organized by topic with solutions.

## Naming Convention

`<topic>-<difficulty>.md` — e.g. `joins-easy.md`, `subqueries-medium.md`, `window-functions-hard.md`

## Difficulty Levels

- **Easy** — single concept, straightforward query
- **Medium** — combines 2-3 concepts, requires some thinking
- **Hard** — complex scenarios, optimization, multiple approaches

Use [exercise-template](../templates/exercise-template.md) to create new entries.
